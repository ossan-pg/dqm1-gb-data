# 技データ

- No：技ナンバー
- ST：技対応作戦番号
- SK：技選択値基本値(技優先度)
- MP：消費MP
- 属：属性ナンバー(参照：耐性データ)
- 変：耐性によって効果がどのように変化するかをしめすタイプです。耐性による命中、ダメージ補正タイプ
- DM1, DM2：DM1 + random(1 + DM2) が基本効果となります
- DE1, DE2：敵の場合の基本効果です
- Type：物理、魔法、ブレス、ダンス
- 溜：力を溜めるで大ダメージの技
- バ：バイキルトでダメージ 2倍の技
- ク：クリティカルが発動する可能性のある技
- 地：高く飛び上がるに効果なしの技
- 正：相手の目をさましてしまう可能性のある技
- 吸：マホキテで MP を吸収できる技
